## 무료 플랜 (오딸) — 상세 기능
- JSON → TypeScript `interface` 또는 `type` 생성
- 최대 nesting 깊이 5
- 배열, optional, null 자동 처리
- 하루 20회 제한 (초과 시 “Basic 업그레이드 유도” 팝업)

## Basic 플랜 (4,900원) — 상세 기능
- 무제한 변환
- Zod 스키마 자동 생성 (`.nullable()`, `.optional()` 정확)
- TanStack Query 타입 자동 생성  
  → `useQuery<GetUserResponse>(...)` 바로 복붙 가능
- React Hook Form + Zod resolver 자동 생성
- 변환 히스토리 최대 50개 저장 + 이름 지정 가능
- 결과물 한 번에 복사 (TS / Zod / Query / Form 4개 탭)

## Pro 플랜 (14,900원) — LLM + 규칙 기반 완전체
| 카테고리                | 구체적인 기능 (복붙하면 바로 되는 코드)                                                                                 |
|-------------------------|--------------------------------------------------------------------------------------------------------------------------|
| AI 리네임 제안          | - usr_nm → userName, mem_no → memberId, crt_dt → createdAt 등 자동 제안<br>- 체크박스로 일괄 적용 / 개별 수정 가능       |
| NestJS DTO 완성         | - `class-validator` + `class-transformer` 데코레이터 자동 부착<br>- `@IsOptional()`, `@IsEmail()`, `@IsDateString()` 등 정확 판단<br>- `@ApiProperty({ description: '…' })` 한국어 설명 자동 삽입 |
| Prisma schema           | - 1:1 매핑 자동 생성<br>- `@id @default(uuid())`, `@unique`, `??` nullable 처리<br>- Relation 필드도 인식 (userId → User) |
| tRPC procedure          | - `router.user.getUser.query(() => …)` 스캡폴드 생성<br>- input Zod, output 타입 자동 연결                             |
| OpenAPI/Swagger 예제    | - 예제 JSON 업로드 → request + response 타입 모두 생성                                                                |
| GitHub 연동             | - Public / Private repo / Gist URL만 붙이면 자동 fetch<br>- 변환 결과 → 자동 PR 생성 (옵션)                           |
| 팀 컨벤션 템플릿        | - 팀별로 camelCase 규칙, 접미사 규칙 저장<br>- “우리 팀은 Id 대신 No 쓴다” 같은 설정 강제 적용                        |
| 히스토리 Diff 뷰어      | - 어제 변환 vs 오늘 변환 차이점 하이라이트 표시                                                                        |
| 우선 처리 큐            | - Pro 유저는 변환 즉시 처리 (무료/Basic은 3~5초 대기)                                                                   |

## Team 플랜 추가 기능

- 변환 결과 팀원 모두 공유 가능
- 팀별 Naming Convention 강제 적용 (위반 시 경고)
- 팀별 Stripe 한 번 결제 → 모든 멤버 Pro 권한
